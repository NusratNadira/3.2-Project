# Federated XAI IDS

## Project Overview
This is a web-based application that allows users to upload datasets, select machine learning models (ANN, CNN, RNN), and view the results.

## Requirements
- Python 3.x
- Streamlit
- TensorFlow
- Scikit-learn
- Pandas
- Matplotlib
- Plotly
